# 16 — UX & Editor Usability Audit Report

> Data: 2026-07-24  
> Projeto: Hr Shoes Commerce  

---

## Auditoria de Experiência do Usuário (UX/UI) no Editor Admin

1. **Intuitividade do Sidepanel**: Organizado em abas claras (*Conteúdo*, *Dados*, *Layout*, *Design*), reduzindo a sobrecarga cognitiva para o lojista.
2. **Edição de Arrays e Subitens (`ArrayBuilder`)**: Suporta expansão/recolhimento tipo accordion, inclusão, exclusão e ordenação drag-and-drop.
3. **Seleção de Mídias e Upload (`MediaUploader`)**: Modal de escolha direta a partir da biblioteca do Supabase Storage ou upload direto do computador.
4. **Troca Rápida de Temas**: Botão de fácil acesso na barra superior permite aplicar um dos 10 presets visuais com modal de confirmação para evitar perda acidental de rascunhos.
5. **Alternância de Viewport (Desktop / Mobile)**: Permite ao lojista visualizar exatamente a experiência responsiva antes da publicação.
